Surface meteorological data at Pontianak (GIF format)

ResourceID
spase://IUGONET/DisplayData/RISH/misc/PON/AWS/aws_pon_gif

Description

The time-series plot of surface meteorological data taken by the Automatic Weather Station (AWS) at Pontianak in West Karimantan, Indonesia (0.00S, 109.37E, 1m MSL), which has been operated on March 16, 2008. This plot includes pressure, temperature, relative humidity, wind velocity, wind direction and rain amount. File names are composed of (year)(month)(day).gif (daily plot), (year)(month).gif (monthly plot) and (year).gif (yearly plot). Local time (UT + 07:00) is used for time data.
