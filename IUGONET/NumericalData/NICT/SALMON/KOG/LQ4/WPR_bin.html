Observation data taken by 1.3GHz transportable wind profiler radar (called LQ4) at Koganei, Japan (binary format)

ResourceID
spase://IUGONET/NumericalData/NICT/SALMON/KOG/LQ4/WPR_bin

Description

The 10-minute average observation data in the lower troposphere (0.1-8 km) taken by the1.3GHz transportable wind profiler radar (called LQ4) at Koganei, Tokyo (35.71N, 139.49E, 80m MSL). The profiles of peak power, Doppler frequency, and spectral width for each beam are stored in the binary file named koglq4-YYYYMMDDhhmm.dat.
