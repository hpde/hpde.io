Surface meteorological data at Shigaraki MU Observatory (CSV format)

ResourceID
spase://IUGONET/NumericalData/RISH/misc/SGK/AWS/aws_sgk_csv

Description

The surface meteorological data taken by the Automatic Weather Station (AWS) at Shigaraki MU Observatory in Shiga Prefecture, Japan (34.86N, 136.11E, 385m MSL). The observation data are stored in the the CSV (Comma Separated Values) file named (year)(month)(day).csv. The numbers of the first line of each file represent date (YYYY/MM/DD), time zone (UT + 09:00), latitude (deg), longitude (deg) and sea level (m), and the second and below are local time (hour:minute:second), pressure (hPa), temperature (degree C), relative humidity (%), zonal and meridional winds (m/s). The value of -999 means missing data.
