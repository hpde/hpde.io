Observation data of the troposphere and lower stratosphere taken by the radiosonde soundings at Shigaraki MU Observatory (tsv format)

ResourceID
spase://IUGONET/NumericalData/RISH/misc/SGK/Radiosonde/radiosonde_sgk_tsv

Description

Editted sounding data at Shigaraki MU observatory in the Shiga prefecture, Japan (34.86N, 136.11E, 385m MSL) since June 27, 2004. The data are also called "EDT data"which converted from the raw data in operating sounding system (DigiCORA), and stored in the TSV (Tab Separated Values) format. The file name of the data are (year)(month)(day)(hour)(minute).tsv which represents launch time of the balloon (in universal time). The line 1-38 represent the file header (map name, sounding set, RS-number, data record lengch, number of data records, max filemap size, data header size, free space in map, status flag (not used), and table for each parameter name, unit, datatype, divisor and offset). The line 39 and below are observation data. The value of -32768 means missing data.
