Observation data of the troposphere and lower stratosphere taken by the radiosonde soundings at Shigaraki MU Observatory (CSV format)

ResourceID
spase://IUGONET/NumericalData/RISH/misc/SGK/Radiosonde/radiosonde_sgk_csv

Description

Radiosonde sounding data at Shigaraki MU observatory in the Shiga prefecture, Japan (34.86N, 136.11E, 385m MSL) since September 26, 1986. The data are converted from .snd (until 2003 and some of 2004 and 2005) or .tsv data (from June 2004), and stored in the CSV (Comma Separated Values) file named (year)(month)(day)(hour)(minute).csv. The file name represents each launch time of the balloon (in local time). The first line of each CSV file represent launching local date, launching local time, time zone (hour), latitude, longitude, sea level and sonde serial number. The second and below are time, height, pressure, temperature, relative humidity and zonal and meridional wind. The value of -999 means missing data. In the CSV file converted from .snd data, the wind data are missing value and valid data are only height, pressure, temperature and relative humidity because the original (.snd) data are obtained with old-type receiver and Vaisala RS-80 type radiosonde transmitters.
