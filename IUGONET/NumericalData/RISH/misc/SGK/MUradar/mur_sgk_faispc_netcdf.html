Field-aligned irregularity (FAI) observation data of the ionosphere taken by the MU radar (netCDF format)

ResourceID
spase://IUGONET/NumericalData/RISH/misc/SGK/MUradar/mur_sgk_faispc_netcdf

Description

The field-aligned irregularity (FAI) observation data in the ionosphere between 80 km and 130 km taken by the MU radar at Shigaraki in the Shiga prefecture, Japan (34.86N, 136.11E, 385m MSL), which has been operated in the special observation mode of the FAI. The data are stored in the netCDF (network Common Data From) file named (year)(month)(day).(observation parameter).nc. The netCDF file includes the range, height, time, radial Doppler velocity, echo power, spectral width and noise level for each beam number and so on. The value of 1.0e+10 means missing data.
