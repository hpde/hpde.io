Field-aligned irregularity (FAI) observation data of the ionosphere taken by the MU radar (CSV format)

ResourceID
spase://IUGONET/NumericalData/RISH/misc/SGK/MUradar/mur_sgk_faispc_csv

Description

The field-aligned irregularity (FAI) observation data in the ionosphere between 80 km and 130 km taken by the MU radar at Shigaraki in the Shiga prefecture, Japan (34.86N, 136.11E, 385m MSL), which has been operated in the special observation mode of the FAI. The data are stored in the CSV (Comma Separated Values) file named (year)(month)(day).(observation parameter).(variable)(beam number).csv. The variable abbreviations are dpl, wdt, dpl, and pn, which mean radial Doppler velocity, echo power, spectral width and noise level for each beam number, respectively. The numbers of the first and second lines of each file represent ranges and heights, respectively, and the third and below are observational local date and time (year/month/day hour:minute) which corresponds to the center time of averaging period and data at each range and height. The value of -999 means missing data. The time resolution depends on each observation parameter. The noise level is the height-averaged value.
