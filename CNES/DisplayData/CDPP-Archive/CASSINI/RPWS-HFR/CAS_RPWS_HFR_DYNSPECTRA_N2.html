CASSINI RPWS L2 Dynamic Spectra (Power on 1 Antenna)

ResourceID
spase://CNES/DisplayData/CDPP-Archive/CASSINI/RPWS-HFR/CAS_RPWS_HFR_DYNSPECTRA_N2

Description

This data set contains 24h dynamic spectra plots of spectral power measured by the RPWS/HFR instrument on one antenna with arbitrary units. Two spectra are available : raw data (top) and background subtracted data (bottom). The subtracted background level is computed for each day at each observation frequency.
Plots are in PDF format.
