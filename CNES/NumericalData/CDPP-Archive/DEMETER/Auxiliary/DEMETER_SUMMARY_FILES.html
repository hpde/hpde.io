DEMETER Science mode summary

ResourceID
spase://CNES/NumericalData/CDPP-Archive/DEMETER/Auxiliary/DEMETER_SUMMARY_FILES

Description

The summary files give, per APID (Application Process IDentifier) from 1129 to 1144 (i.e. per data type), the start and the end times of the scientific modes for each Demeter orbit.
The filename is of the type DMT_SUMMARY_APID_aaaa_start_orbit_end_orbit_start_time_end_time where aaaa is the APID number from 1129 to 1144.
Data record are arranged chronologically.
One record containing :
Half-orbit number, Mode (Survey, Burst or All), Data start time, Data end time
Data in the records are separated by blanks.
