CLU3 WHISPER Event information under natural mode

ResourceID
spase://CNES/NumericalData/CDPP-Archive/Cluster-3/WHISPER/CLU_C3_WHI_NATURAL_EVENT

Description

This Event Table shows the telecomand status of the WHISPER sounder on the Cluster-3 spacecraft when passive (i.e., when measuring natural plasma waves).
For each time interval, the following information is provided :
time_tags__C1_CT_WHI_NATURAL_EVENT : The time interval
fft_size__C1_CT_WHI_NATURAL_EVENT : The number of frequency bins of the Fast Fourier Transform
Receiving_Antenna__C1_CT_WHI_NATURAL_EVENT : The receiving antenna used, Ey or Ez
Average_Number__C1_CT_WHI_NATURAL_EVENT : The number of averaged individual spectra. Individual spectra are produced at the rate of one every 13.33 ms
Gain_Command__C1_CT_WHI_NATURAL_EVENT : The gain command number :
0: fixed gain +12dB
1: fixed gain +24dB
2: fixed gain +36dB
3: fixed gain +36dB
4: automatic gain+24/+12 dB
5: automatic gain +24/+12 dB
6: automatic gain +36/+24 dB
7: automatic gain +36/+24 dB)
Averaged_Spectrum_Select_Rate__C1_CT_WHI_NATURAL_EVENT: Output rate of the averaged spectra
Energy_Overflow_Averaging_Flag__C1_CT_WHI_NATURAL_EVENT :
Averaging status:
0: OFF
1: ON
Compression_Methode_Required__C1_CT_WHI_NATURAL_EVENT:
0 if the exponent size is adjusted
1 if the exponent size is fixed to 5
Compressed_Data_Size__C1_CT_WHI_NATURAL_EVENT :
Compressed data size:
0 means 8 bits,
1 means 6 bits
