ARCAD3 TBF Waveforms : E and B fields

ResourceID
spase://CNES/NumericalData/CDPP-Archive/ARCAD-3/TBF/ARC_TBF_WFO

Description

This data set contains ARCAD3 TBF waveforms, E and B fields.
The TBF experiment onboard the AUREOL3 satellite (ARCAD3 project) was designed to measure two electric and three magnetic wave field components in the range from 10 Hz to 16 kHz. Two telemetry systems were available on AUREOL3, high bit-rate real-time telemetry (TMF) and tape-recorded telemetry (TMS). The Waveform data are related to the TMF one. The ground stations mostly used are Tromsö (Norway) and Terre-Adélie (French Antartica).
Two electric and three magnetic components were digitized with a sampling frequency of 5000 Hz. The electric field is measured using spherical probes located at the tip of booms and the alternative magnetic field is measured using a search-coil instrument.
The set of data contains : general information (date, time, orbit, telemetry, ...), orbit parameters (latitude, longitude, ...), attitude parameters (angles, matrices for coordinate system changes), waveform data (component, unit, gain, amplitude).
TBF Waveforms data use
TBF data presented in the 'Waveform' archive have to be validated prior to any interpretation. Questionable electric and magnetic field data can be due to :
telemetry noise,
calibrations performed every 128 seconds,
possible saturation of the input electronics (ULF turbulence),
wrong gain variations,
gain variation inside a 960 ms data frame.
The transfer functions used for the conversion in physical values do not allow any utilization of the data below 5 Hz.
Concerning the ancillary data, for several sets, the orbit parameters can be equal to zero even the related flag is correctly set (OK_AUXI).
