ARCAD3 ISOPROBE probe sweeps

ResourceID
spase://CNES/NumericalData/CDPP-Archive/ARCAD-3/Isoprobe/ARC_ISO_BALAYAGE

Description

This data set contains ARCAD-3 Isoprobe probe sweeps (sweep information, amplitude and phase).
The logarithmic amplitude and the phase values of the mutual impedance are given versus the frequency (log(|Z|) and PHI(Z)). Four basic sweeps are stored : B1 active sweep from 100 kHz to 9675 kHz with the DC configuration, B2 active sweep from 100 kHz to 9675 kHz with the DL configuration, B3 passive sweep from 100 kHz to 9675 kHz with the DL configuration and B4 active sweep from 100 kHz to 1587.5 kHz with the DL configuration. The frequency resolution depends on the telemetry type (TMF, TMS4, TMS30 and TMS120) and on the operating mode (BAFA, BAFI, BAFP and BAFC). A fifth sweep (status sweep) is recorded when the experiment is set in a high resolution mode (extremum tracking).
