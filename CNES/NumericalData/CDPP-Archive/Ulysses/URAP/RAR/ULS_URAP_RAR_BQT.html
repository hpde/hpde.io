ULS-URAP RAR_BQT : Plasma Parameters (quasi-thermal noise method)

ResourceID
spase://CNES/NumericalData/CDPP-Archive/Ulysses/URAP/RAR/ULS_URAP_RAR_BQT

Description

This data set contains daily files of ULYSSES/URAP radio data. These files contain plasma parameters (mainly the electron density and temperature) computed by the plasma quasi-thermal noise method from 128 s radio spectra measured on the 1.25-48.50 kHz receiver connected to the thin wire 2x35m electric dipole.
