GEOS-1 Direct Current Magnetic Field Component Level-2 (RFF)

ResourceID
spase://CNES/NumericalData/CDPP-Archive/GEOS-1/S-331/GEOS1_MAG_VTL2_RFF

Description

This data set contains the calibrated DC magnetic field from the GEOS-1 S-331 fluxgate magnetometer given in VDH Coordinate System. Time resolution is 88.064s.
The data format is RFF (Roproc File Format ftp://ftp.lpp.polytechnique.fr/robert/keep/Biblio_et_CV/Working_documents/2015_Robert_The_Roproc_File_Format_V2p3.pdf)
