GEOS-1 Ultra Low Frequency Magnetic Field Components Level-1 (RFF)

ResourceID
spase://CNES/NumericalData/CDPP-Archive/GEOS-1/S-300/GEOS1_UBF_VTL1_RFF

Description

This dataset contains magnetic field components from GEOS-1 S-300 wave experiment (search coil magnetometer) given in Instrument Spinning System in TM counts. Time resolution is 43 ms.
The data format is RFF (Roproc File Format ftp://ftp.lpp.polytechnique.fr/robert/keep/Biblio_et_CV/Working_documents/2015_Robert_The_Roproc_File_Format_V2p3.pdf)

Bx, By, Bz: Magnetic field
Gx, Gy, Gz: adaptative gains that take values 5, 25 or 125
Dx, Dy: despin signals corresponding To the continuous magnetic field rotation

The conversion to Volt units is given by:
Bx(V) = -(Bx5.1/255.0 - 2.5)/Gx
Dx(V) = (Dx5.1/255.0 -2.5)/10.0
