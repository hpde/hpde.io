CLU2 WHISPER passive electric power spectral density when in sounding mode

ResourceID
spase://CNES/NumericalData/CDPP-Archive/Cluster-2/WHISPER/CLU_C2_WHI_PASSIVE_ACTIVE

Description

This dataset contains the electric power spectral density determined on Cluster-2 during passive sweeps of the WHISPER sounder which occur when the sounder is nominally active.
These passive sweeps are interspersed with active sweeps to assist the identification of plasma resonances by comparison of active and passive sweeps adjéacent in time.
These sweeps are certainly not active sweeps, but neither are they all 100% free of echoes from nearby active sweeps.
The following support parameters are provided :
Receiving antenna used, Ey or Ez
Spectral Frequencies between 1.953 and 79.915 kHz, that is, the frequencies corresponding point-to-point to the 512 elements of the fixed size spectrum
Frequency Table Number between 0 and 127 Time interval between two successive emission pulses in units of 40/3 (= 13.3) ms :
0: 1 unit (13.3 ms)
1: 2 units (26.6 ms)
2: 3 units (40 ms)
3: 5 units (66.6 ms)
4: 8 units (106.6 ms)
5: 40 units + 2106.66 ms waiting + 256 units + 2446.66 ms waiting (8.5 s)
6: 6 units + 45 ms waiting (125 ms)
7: 14 units + 63.34 ms waiting (250 ms).
Delay between the acquisition intervals used to form the passive spectrum information and the acquisition intervals used to form the correponding active spectrum information
Number of gain changes during the acquisition
First part of the compressed value which is a fixed point integer defined in the compression algorithm
Second part of the compressed value
