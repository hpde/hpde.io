MMS-4 FPI DES Level-2 Burst electron distribution functions (CDF)

ResourceID
spase://CNES/NumericalData/CDPP-Archive/MMS-4/FPI/DES/MMS_FPI_4_L2_BRST_DES_DIST

Description

This dataset provides the full electron distribution functions at the BURST cadence (30 ms). The data is provided in units of phase space density. The reference frame is related to the instrument frame, and is called near-GSE.
For more information on the FPI instruments, data set, and user guide, please see:
MMS Science Data Center (https://lasp.colorado.edu/mms/sdc/public/datasets/fpi/)
MMS FPI Data Users and Products Guide (https://lasp.colorado.edu/galaxy/display/MFDPG/MMS+FPI+Data+Users+and+Products+Guide)
FPI Release Notes (https://lasp.colorado.edu/mms/sdc/public/datasets/fpi/mms-FPIReleaseNotes-220217.pdf)
