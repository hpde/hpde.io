VENUS EXPRESS orbit data (cruise phase, GSE frame, 1 min resolution)

ResourceID
spase://CNES/NumericalData/CDPP-Archive/VEX/Ephemeris/VEX_ORBIT_GSE

Description

This data set contains VENUS EXPRESS position and velocity in GSE frame, 1 min resolution, during the cruise phase.

Each ASCII file represents one day.
The coordinate system used is GSE (Geocentric Solar Ecliptic).
Data record are arranged chronologically.
One record each minute containing :
-Time-tag, X.frame (km), Y.frame (km), Z.frame (km), Vx.frame (km/s), Vy.frame (km/s), Vz.frame (km/s)
-Data in the records are separated by commas and blanks.
Caution: For the first line, the velocity is NaN (Not a Number) initialized.
