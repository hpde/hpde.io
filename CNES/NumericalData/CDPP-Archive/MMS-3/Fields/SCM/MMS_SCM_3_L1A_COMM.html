MMS-3 SCM Search Coil Magnetometer Level-1a Commissioning Waveform (CDF)

ResourceID
spase://CNES/NumericalData/CDPP-Archive/MMS-3/Fields/SCM/MMS_SCM_3_L1A_COMM

Description

This data set contains MMS-3 SCM uncalibrated AC magnetic field waveforms (in telemetry unit) and in the sensor frame.
The data in the (spinning at 20 s) sensor frame are sampled at (survey scsrvy=32 S/s, brst scb=8192 S/s, schb=16384 S/s).
For this case of commissioning data, the sample frequency can be different from the nominal modes, and is indicated in the name of the CDF files (example mms1_scm_comm_l1a_sc128_20150316_v1.1.0.cdf where the frequency corresponds to 128 Hz (sample per second)).
