VIKING V4 : E and B Waves and Plasma Density (V4L/V4H)

ResourceID
spase://CNES/NumericalData/CDPP-Archive/Viking/V4/VIKING_V4_DATA

Description

This data set contains electric and magnetic fields and plasma density from the VIKING V4 waves experiment.

The access to the VIKING-V4 high resolution data can be performed either from the orbit number or from the time interval of interest. This dataset contains 1104 not consecutive orbits from #37 (1986-02-28 14:49:12) to #1653 (1986-12-19 09:15:24).

The data are indented per periods of 15 min. Inside these 15 min, the data are organized in structures (called BU) synchronized on the sweeping time of the SFAs, which usually lasts 2.4 s (except when the RS mode is on : 4.8 s). Inside each BU, the data correspond to a constant configuration of the wave experiment. This is automatically true for the V4H experiment. If this is not verified for the V4L experiment then these data contain filling values after the time change of configuration.
