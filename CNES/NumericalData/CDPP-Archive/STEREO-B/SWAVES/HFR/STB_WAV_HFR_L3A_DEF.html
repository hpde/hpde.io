HFR L3A STEREO-B 'DEF' High Frequency Receiver (125 - 16025 kHz) - Goniopolarimetry parameters

ResourceID
spase://CNES/NumericalData/CDPP-Archive/STEREO-B/SWAVES/HFR/STB_WAV_HFR_L3A_DEF

Description

This data set contains definitive goniopolarimetry parameters are derived from STEREO-B WAVES HFR (High Frequency Receiver) high resolution data (125 - 16025 kHz).

Level 3a data are computed asssuming: (i) incoming wave properties do not change over three successive antennas-pair measurements, (ii) signals measured by an antenna are the same on the two receiver channels, (iii) radio sources have an uniform brightness distribution.
