Wind Waves TNR L2 Temporal Series Data

ResourceID
spase://CNES/NumericalData/CDPP-Archive/Wind/WAVES/WIND_WAVES_TNR_L2_ACE

Description

Wind Waves Thermal Noise Receiver Temporal Series
These are calibrated data presented as a temporal series. Each record consists on a unique spectrum (given in µV^2 / Hz) made with 96 frequencies.
In modes 0, 1 and 4 the number of channels per band is increased to 32 by interpolation. The interpolated frequencies are located by the odd indexes.
In mode 4, spectra issued from the TNR B are ignored. In mode 2 and 3, data are not altered.
If the cycle is in state fixed tune (event_state = 1) , then only 32 values are defined. With respect to the band sweeped, the following values are defined:
Band: Index of defined values
A : 0 to 31
B : 16 to 47
C : 32 to 63
D : 48 to 79
E : 64 to 95
If the cycle is in state ABCDE (event_state=2), then band B and D are ignored, while in state ACE (event_state=3) all data are available.
