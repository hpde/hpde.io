Wind Waves TNR_60S : 60-second Averaged Data (Range 4 - 256 kHz)

ResourceID
spase://CNES/NumericalData/CDPP-Archive/Wind/WAVES/WIND_WAVES_TNR_60S

Description

This data set contains WIND WAVES Thermal Noise Receiver 60-second Averaged Data (Range 4 - 256 kHz, number of channels 96)
Experimental data have been re-processed at LESIA from 60-second averaged data produced by the GSFC team at NASA. They are presented as temporal series. Each record consists on an averaged spectrum (given in µV^2/Hz).
