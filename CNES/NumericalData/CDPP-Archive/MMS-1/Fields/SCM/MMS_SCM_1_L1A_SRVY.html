MMS-1 SCM Search Coil Magnetometer Level-1a Survey Waveform (CDF)

ResourceID
spase://CNES/NumericalData/CDPP-Archive/MMS-1/Fields/SCM/MMS_SCM_1_L1A_SRVY

Description

This data set contains MMS-1 SCM uncalibrated AC magnetic field waveforms (in telemetry unit) in survey mode and in the sensor frame.
These raw uncalibrated (telemetry units) AC magnetic field waveform data in the (spinning at 20 s) sensor frame are sampled at (survey scsrvy=32 samples/s, brst scb=8192 samples/s, schb=16384 samples/s).
