Description

This dataset contains definitive orbit ephemeris used to create L2 data from L1b data (probe position, speed and mass)
GTP files are produced when problems occurred on original files. These are the correct files to be used in order to generate level 2 files.
For more information, please have a look at the SCM Data Products Guide
