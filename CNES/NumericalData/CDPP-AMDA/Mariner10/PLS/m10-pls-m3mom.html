ResourceID
spase://CNES/NumericalData/CDPP-AMDA/Mariner10/PLS/m10-pls-m3mom

Description

This dataset contains parameters of the electron distribution (density,
temperature) derived from the plasma science experiment on Mariner 10
during the first Mercury encounter. This file contains Maxwellian fits
to the distribution halo, and two cores, plus a moment estimate. If the
distribution does not have a halo, or only has a single core, then the
halo or core2 columns will contain missing data flags.
