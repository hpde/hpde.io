These data contain persistent noise signals at 0.2, 8, 16 Hz, and associated harmonics. This noise was not observed during ground testing of the instruments and is anticipated to be interference arising from the spacecraft. Removal of these signals is at the user's discretion. There are errors in some of the Cassini MAG data in this data set due to incorrect handling of leapseconds. The errors are as follows: * 2012-07-01 00:00:00.000 - 2015-07-01 00:00:00.000 - Error : +1 s * 2015-07-01 00:00:00.000 - 2017-01-01 00:00:00.000 - Error : +2 s * 2017-01-01 00:00:00.000 - End of Mission - Error : +13 s The time tags can be corrected by subtracting the error from current time values. This error was created during the conversion of the data from binary to ASCII performed at the PDS/PPI node. It does not exist in the original binary data files available from the Cassini MAG team. The PPI node will correct this error in it's data holdings at a future date. Some versions of these data were determined to contain timing errors, including both backwards and repeated times. These timing errors have not been corrected in the data files.
