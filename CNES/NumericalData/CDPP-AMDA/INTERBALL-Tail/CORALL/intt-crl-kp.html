Interball Tail - Corall - key parameters

ResourceID
spase://CNES/NumericalData/CDPP-AMDA/INTERBALL-Tail/CORALL/intt-crl-kp

Description

CORALL is the ion spectrometer with energy range 30eV - 25keV onboard INTERBALL/TAIL satellite. It has a fan-shape aperture wich plane has a sunward spin axis of spacecraft. The full distribution function is obtained for one spacecraft revolution. So for every revolution plasma parameters ( V, T, N) can be calculated.
