Polar TIDE Level-Zero Processing Software

ResourceID
spase://SMWG/Service/POLAR/TIDE/LZsoftware

Description

level-zero processing software provides several analysis and display options with an extensive list of processing options. The
TIDE team has provided websites that allow users to process data from the web and then download the processed datafiles and
plots, see the Custom Plot and Data Dump links on one of the TIDE Home Pages. The software (C and IDL) can be downloaded
and installed locally. There is a data download page that can be used to get the level-zero data and the required ancillary data (Polar
orbit and attitude, extracted EFI spacecraft potential, and ion masks). NOTE: The only way to read a TIDE level-zero file is with the
processing software. See the TIDE Documentation link on one of the TIDE Home Pages for documentation of the processing
software and the input file data formats.

Description

TIDE Level-Zero Processing Software Documentation webpage contains links to webpages that discuss all aspects of the TIDE
level-zero processing including the required input and all the processing options (see Parameter File Creation and Major Processing
Options). Most of the processing options can be run from the Custom Plot and Data Dump links on one of the TIDE Home Pages.
Use of these webpages is discussed in the documentation. The stand-alone IDL version of the software is also discussed in the
documentation.
