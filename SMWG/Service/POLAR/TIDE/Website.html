Description

TIDE Website contains information about the instrument, its operation, its level-zero data, and its processing software. Users can
view spectrogram summary plots and interactively process TIDE level-zero data for the entire Polar Mission (03/28/96 through
04/16/08). Only the major webpages are discussed below. There are several other links on the home page. There are mirror sites
at the National Space Science and Technology Center (NSSTC) and GSFC.

GSFC TIDE Summary Plot Gifwalk
URL
http://tide.gsfc.nasa.gov/summary_plots.shtml/
Description

TIDE Summary Plot Gifwalk lets the user scan through TIDE high time resolution spectrogram plots for the entire Polar mission.
The user selects the ion type: Total (Ion), H+, O+, or He+; the year, month, and day to start the viewing; the spectrogram type: half-orbit,
perigee, 6-hr PSI operations, or plasmasphere passes; and then presses Update Date/Type/Ion. Once the gif image is displayed,
there are arrow keys that allow the user to move forward and backward in time. The user can download the gif by right click / save
picture as or click on Download the pdf version to see and download the higher quality pdf plot. There is a TIDE Website navigation
bar at the bottom of the webpage. The user can zoom in on an area of interest by clicking on the image at that point. That takes the
user to the High Time Resolution Spectrogram Custom Plot Webpage, where after changing any default processing options and
Submitting Request, a new spectrogram plot will be created.

GSFC TIDE View Thumbnails
URL
http://tide.gsfc.nasa.gov/display_thumbnails.shtml
Description

TIDE Thumbnails webpage provides a way to view several TIDE summary spectrograms quickly. Just specify the time range
and select the ion and the spectrogram type (either half-orbit or perigee) and submit request. Thumbnails of each spectrogram in
the time range are displayed 8 per row. Move the mouse over a thumbnail to get the start time of the plot. Click on the thumbnail to
view the full size gif and get a link to the pdf.

GSFC TIDE Custom Plots
URL

Description

TIDE Custom Plots website allows users to process TIDE level-zero data for several different plot types including 1-D and 2-D
spectrograms, phase space density velocity distributions, moments (density, velocity, and temperature), phase space density versus
velocity, chromograms, and Polar orbit. Once the processing is complete, the user can download the plots (gif, postscript, and/or pdf).
If the plot is associated with a text file containing the results (moments, velocity distributions, and psd vs. velocity), then there will be
a download link for that file. See http://tideuviRA.nsstc.nasa.gov/tide/Documentation/ProcessingSoftware/ProSoft.html or
http://tide.gsfc.nasa.gov/Documentation/ProcessingSoftware/ProSoft.html for details about each processing type.

Description

TIDE Data Dumps webpage allows users to download text files or plots of certain types of data including
engineering/housekeeping text dump or summary plots, instrument parameters, magnetic field location, spin header information,
major frame byte dump, and dumps of raw and processed counts/flux. Once the processing is complete, the user can download
the requested datafile or plots. See http://tideuviRA.nsstc.nasa.gov/tide/Documentation/ProcessingSoftware/ProSoft.html or
http://tide.gsfc.nasa.gov/Documentation/ProcessingSoftware/ProSoft.html for details about each plot type.
