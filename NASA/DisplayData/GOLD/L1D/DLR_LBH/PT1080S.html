Gold Level 1D German Space Agency Disk LBH Plots

Eastes, Richard (2025). Gold Level 1D German Space Agency Disk LBH Plots [Data set]. Space Physics Data Facility. https://doi.org/10.48322/fhp7-r930 .

Note: Proper references, including those in BibTex or other formats , should include the "Accessed on date" as shown above to identify the version of the resource being cited in a given publication.

ResourceID
spase://NASA/DisplayData/GOLD/L1D/DLR_LBH/PT1080S

Description

Level 1D Lyman-Birge-Hopfield (LBH) files contain a thumbnail image (PNG) for individual disk scans.
There are individual thumbnail files created for total LBH radiance (kR). Separate
files are created for Channel A and Channel B data. A new L1D file is generated for each new L1C file.
These can correspond to either the Northern or Southern hemisphere.
