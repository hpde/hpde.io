ACE Magnetic Reconnection Exhaust List

ResourceID
spase://NASA/Catalog/ACE/MagneticReconnectionExhaust

Description

A list of magnetic reconnection exhausts observed by the Advanced Composition Explorer (ACE) spacecraft were identified using the unedited SWEPAM data set available at Los Alamos. Some of these events are not obvious in the level 2 SWEPAM data because of the editing process used in producing the level 2 data set.
