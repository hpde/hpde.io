STEREO/EUVI Event Catalog (>C1 GOES class or >12 keV RHESSI)

ResourceID
spase://NASA/Catalog/STEREO/SECCHI/EUVI/EventCatalog

Description

This is a catalogue of flare/CME events observed with EUVI when the spacecraft separation angle is small to moderate when classical stereoscopy is feasible. The list contains also the peak fluxes in soft X rays and hard X rays.
EUVI Classification:
A=Ahead only without Behind data B=Behind only without Ahead data
I=Impulsive time profile (peak in coincidence with SXR)
P=Postflare loop or arcade (delayed emission after SXR peak)
D=Dimming (EUV flux decreases after SXR peak)
E=Eruptive filament
O=Occulted for A (if flare position is east) or B (if west)
T=Test images (or poor image quality)
W=Waves and oscillations
CME reported: L=LASCO/SOHO, S=SECCHI/Cor-1,Cor-2
