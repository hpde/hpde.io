(2022). STEREO Stream Interaction Region List [Data set]. NASA, STEREO Science Center. https://doi.org/10.48322/s7mq-ns41 (CiteAs link). Accessed on .

ResourceID
spase://NASA/Catalog/STEREO/SIRs.List

Description

This catalog has a list of stream interaction regions
(SIR's) observed by STEREO-A and/or STEREO-B between March 2007 and March

  1. For each of the ~145 STEREO A or B observations, the following
    information is given: event start time, time of the peak pressure, event
    end time, maximum values of pressure, magnetic field magnitude, proton
    density and flow speed, and minimum flow speed. It is expected that this
    list, given in both PDF and Excel formats, will be updated periodically.
