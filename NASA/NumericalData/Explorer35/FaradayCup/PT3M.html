Explorer 35 3-min solar wind plasma data

ResourceID
spase://NASA/NumericalData/Explorer35/FaradayCup/PT3M

Description

This data set has 3-min resolution solar wind plasma parameters (flow speed and direction angles, ion thermal speed and density) from the MIT Faraday Cup on selenocentric Explorer 35 (AIMP 2, IMP E) spacecraft. There are geocentric and selenocentric spacecraft position data in the data records. The data are in annual ASCII files.
