Description

PADRE MeDDEA Level0 Spectrum FITS data are uncalibrated hard X-ray spectrum data obtained by the Measuring Directivity to Determine Electron Anisotropy (MeDDEA) instrument on the Solar Polarization and Directivity X-Ray Experiment (PADRE) CubeSat observatory. The Level0 spectrum files contain the same spectrum data as the raw MeDDEA binary data, but converted to FITS format for more convenient analysis. Each raw spectrum data file generates a single Level0 spectrum FITS file. The spectrum product is provided regularly and contains summary spectra created for detector pixels at regular cadence, normally every 10 seconds, with 512 energy bins. These data support investigations of the directivity of solar flare hard X-ray emission and the angular distribution of flare-accelerated electrons.

Photon Spectrum Data, FITS Extension 1
ParameterKey
SPEC
Description

FITS extension 1, named SPEC, contains the Level 0 photon spectrum data as a compressed three-dimensional image array. The first dimension contains 512 uncalibrated spectral channels, the second dimension contains 24 detector pixels, and the third dimension contains successive spectra acquired during the time interval covered by the file.

Parameter #2
Name
Spectrum Packet Data, FITS Extension 2
ParameterKey
PKT
Description

FITS extension 2, named PKT, is a binary table containing packet-level information associated with the spectra in FITS extension 1. Each row provides the packet time, packet subsecond clock, ASIC identifiers and electronic channel numbers for the 24 detector pixels, and packet sequence number. Its columns are pkttimes, pktclock, asic, channel, and seqcount.
