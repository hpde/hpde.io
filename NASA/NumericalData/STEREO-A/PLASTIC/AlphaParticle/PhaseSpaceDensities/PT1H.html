Description

This data set presents STEREO-A PLASTIC PickUp Ion, PUI, He⁺² phase space densities, PSD, in the spacecraft frame of reference. The data set provides 1 hr averages of He⁺², phase space density as a function of V/Vsw in 51 velocity ranges, where V and Vsw are the PUI particle speed and bulk speed of solar wind protons, respectively. These data are delivered to the public domain on a regular basis. Efforts are made to include the latest known calibration and background determinations, however, these are expected to undergo revision. We therefore suggest that users regularly return to this page, and make sure to get the most recent revision. If the STEREO PLASTIC data are used in presentations or publications, we strongly suggest that that the authors contact A.B. Galvin, STEREO PLASTIC PI, at toni.galvin@unh.edu and Dr. B. Klecker at berndt.klecker@mpe.mpg.de to ensure that the latest release of the data is utilized. Also, please acknowledge the STEREO PLASTIC Investigation, A.B. Galvin and NASA Grants NNX15AU01G and 80NSSC17K0556. For reporting purposes, we request bibliography information for any publication, etc., using these data. Please send information on the use of these data to A.B. Galvin. If you have questions regarding the data formats, please contact the STEREO PLASTIC Data System Manager: Lorna Ellis at lorna.ellis@unh.edu.
