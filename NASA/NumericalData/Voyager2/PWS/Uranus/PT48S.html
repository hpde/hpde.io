Voyager 2 Uranus Plasma Wave Spectrometer (PWS) Resampled Spectrum Analyzer, Version 1.0, 48 s Summary Data

Decker, Robert B.; Krimigis, Stamatios M. (2026). Voyager 2 Uranus Plasma Wave Spectrometer (PWS) Resampled Spectrum Analyzer, Version 1.0, 48 s Summary Data [Data set]. Space Physics Data Facility; Planetary Data System.

ResourceID
spase://NASA/NumericalData/Voyager2/PWS/Uranus/PT48S

Description
  • Data Set Overview
  • =================

This Data Set consists of 48 s calibrated, averaged Wave Electric Field Intensities from the Voyager 2 Plasma Wave Receiver Spectrum Analyzer obtained in the Vicinity of the Uranian Magnetosphere. For each 48 s Interval, a geometric average Field Strength is determined for each of the 16 Spectrum Analyzer Channels whose Center Frequencies range from 10 Hz to 56.2 kHz and which are logarithmically spaced in Frequency, four Channels per Decade. The Time associated with each Set of Averages is the Beginning of the Averaging Interval. Averages are stored in Units of V/m. During Data Gaps where complete 48 s Intervals are missing, no Entries exist in the File, that is, the Gaps are not Zero-filled or Tagged in any other way. Additional Information about this Data Set and the Instrument which produced it can be found elsewhere in this Catalog. An Overview of the Data in this Data Set can be found in (Gurnett et al., 1986) and a complete Instrument Description can be found in (Scarf and Gurnett, 1977).
