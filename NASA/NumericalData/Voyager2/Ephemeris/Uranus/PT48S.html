Voyager 2 Uranus Ephemeris in Uranian U1 Coordinates, Version 1.0, 48 s Summary Data

ResourceID
spase://NASA/NumericalData/Voyager2/Ephemeris/Uranus/PT48S

Description
  • Data Set Overview
  • =================

This Data Set consists of Voyager 2 Uranus Encounter Ephemeris Data relative to the Planet in Minus U1 Coordinates.

The U1 or Uranus West Longitude Coordinate System is a Planetocentric System fixed to Uranus, which is rotating with a 17.24 h Period.

The Prime Meridian of this System is defined such that at 1986-01-24T18:00:00.000 the Voyager 2 Spacecraft was at 302° West Longitude (58° East Longitude).

For the PDS Archive Data Set, the original Data Archive has been converted to an East Longitude (right-handed) Coordinate System Minus U1 (-U1). This is achieved by subtracting the U1 Longitude from 360°. The Data are provided in Units of Uranian Radii, which was taken to be equal to 25,600 km by the Voyager Project. Angles are given in Degrees. Both Cartesian and Spherical Coordinates are provided.

  • Cartesian Coordinates
  • =====================
  • X lies in the Equatorial Plane of Uranus, positive away from the Planet, and through the Prime Meridian at the Reference Epoch
  • Y completes the right-handed Set, Y = X ⨯ Z
  • Z points along the Spin Axis of Uranus, positive in the Southern Hemisphere in orbital Ecliptic Coordinates

The Cartesian Coordinate System does not rotate with the Planet.
