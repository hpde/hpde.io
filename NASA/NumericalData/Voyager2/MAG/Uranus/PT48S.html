Voyager 2 Uranus Magnetic Field in Uranian U1 Coordinates, Resampled, Version 1.0, 48 s Summary Data

ResourceID
spase://NASA/NumericalData/Voyager2/MAG/Uranus/PT48S

Description
  • Data Set Overview
  • =================

This Data Set includes Data from the Low Field Magnetometer (LFM) in the Solar Wind during the Uranus Encounter. The Encounter Data (1986-01-24T07:00:00 to 1986-01-25T04:00:00) have been averaged from the 9.6 s Averages to a 48 s resampled Rate and are provided in the Uranus Longitude (U1) Coordinate System. Magnetometer Data in the Solar Wind are given in Heliographic Coordinates and are also available in a separate Data Set.

The Data Set consists of the following Columns:

    1. ctime - Decimal Seconds since 1966-01-01T00:00:00.000
    1. pdstime - ISO Standard Time Format
    1. Spacecraft Clock Value - m65536
    1. Spacecraft Clock Value - mod60
    1. Spacecraft Clock Value - fds_line
    1. mag_id - Magnetometer ID: 1=LFM, 2=HFM
    1. Br - Radial Component
    1. Bphi - Longitudinal Component
    1. Btheta - Colatitudinal Component
    1. Bmag - Magnitude of the averaged Magnetic Field Components
    1. avg_Bmag - Average of the Magnitude of the raw Components
    1. lambda - Magnetic Longitude, lambda=atan(Bt/Br)
    1. delta - Magnetic Latitude, delta=asin(Bn/avg_Bmag)
    1. rms_br - Br Pythagorean RMS Deviation of the Component Average
    1. rms_bt - Bt Pythagorean RMS Deviation of the Component Average
    1. rms_bn - Bn Pythagorean RMS Deviation of the Component Average
    1. npts - Number of Points in the Average
    1. Data Flag - A Character String which indicates Software or Spacecraft Hardware Intervention which reduces Confidence in the Data, Null Flags represent "good" Data

The U1 Coordinate System is a Non-inertial Spherical Coordinate System that rotates with Uranus with a Period of 17.24 h. The Magnetic Field Components in the U1 Coordinate System are:

  • Br - Radial Magnetic Field Component, along the Uranus to Spacecraft Line, positive away from Uranus
  • Bphi - East Longitudinal (U1) Magnetic Field Component
  • Btheta - Colatitudinal (U1) Magnetic Field Component

An ancillary Data File containing the derived Spacecraft Magnetic Field is provided with these Data at a 48 s Sample Rate. These Data provide Zero Level Offsets used in the Data Processing. Spacecraft Field Data is provided in Spacecraft Coordinates.
