Voyager 2 Uranus Plasma Science (PLS) Derived Ion Fits, Version 1.0, 48 s Summary Data

ResourceID
spase://NASA/NumericalData/Voyager2/PLS/Uranus/PT48S

Description
  • Data Set Overview
  • =================

This Data Set contains the total Ion Density obtained from Voyager 2 PLS Data (Voltage Range 10 eV/Q to 5950 eV/Q) at Uranus by fitting the measured Spectra with isotropic Maxwellian Distributions. It is a Subset of the Data Set VG2-U-PLS-5-RDR-IONFIT-48SEC. Up to three isotropic Proton Components (warm, hot, and intermediate) were used to match the Observations. These were then summed to give total Ion Density. This Determination does not include possible Plasma outside the PLS Energy Range; Limits on a cold Component can be derived in the Charging Region from 19:39 to 21:58 on Day 24. It is found that if a cold Component exists it will increase the total Ion Density by at most a Factor of Two. Uncertainties in Plasma Parameters are a Factor of Two in the Charging Region, less than 20% elsewhere. A complete Description of this Data Set is given in (McNutt et al., 1987) and (Selesnick and McNutt, 1987). Data Format: the first six Columns are Time Tags (6I4), the last Column is the total Ion Density in cm^-3 (E11.3). Flag Values of 1.0E32 indicate that the Parameter could not be obtained from the Data using the standard Analysis Technique. Additional Information about this Data Set and the Instrument which produced it can be found elsewhere in this Catalog. An Overview of the Data in this Data Set can be found in (McNutt et al., 1987) and (Selesnick and McNutt, 1987) and a complete Instrument Description can be found in (Bridge, 1977).
