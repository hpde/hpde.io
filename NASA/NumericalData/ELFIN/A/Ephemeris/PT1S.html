Description

The Electron Losses and Fields Investigation (ELFIN) Mission is a Space Weather Mission using three Scientific Instruments in a 3U+ CubeSat. The Instruments measure Wave and Particle Data. This Data Product is a Probe State File that contains CubeSat Position, Velocity, Spin Attitude, Spin Rate, and Spin Phase Data.
