PUNCH Level 3 Low-noise Polarized Mosaic Data in the MZP system

Southwest Research Institute (2026). PUNCH Level 3 Low-noise Polarized Mosaic Data in the MZP system [Data set]. Solar Data Analysis Center.

ResourceID
spase://NASA/NumericalData/PUNCH/NFI_WFI_Mosaic/Level3/PAM/PT32M

Description

PUNCH level 3 low-noise polarized data in the MZP (Minus, Zero, and Plus for –60°, 0°, and +60°) coordinate framework, proceeded with F-corona subtraction applied, starfield signal unremoved. This dataset is a member of the Collection of PUNCH Level-3 Science Image dataset. Per instructions from the PUNCH team, reference to the entire Collection (including the DOI, see below) should be used when this and other member datasets in the Collection are needed for citation purposes:

Southwest Research Institute (2025). PUNCH NFI-WFI Level-3 Science images [Collection]. Solar Data Analysis Center. https://doi.org/10.48322/nnv7-bn21. Access date (assigned on date of landing page access).
