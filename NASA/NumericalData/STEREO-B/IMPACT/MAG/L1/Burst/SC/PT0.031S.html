IMPACT Magnetic Field Instrument Status Flag

Caveats
IMPACT Magnetometer Status Flag encoded values: bi to bit 3: Magnetometer error counter, bi to bit 7: Magnetometer command counter, bit 8: Spare, bit 9: First, bit 10: Time, bit 11: Interface parity error flag (0=OK, 1=Fault), bit 12: Interface timeout error flag (0=OK, 1=Fault), bit 13: Analog to Digital Converter, ADC, calbration flag (0=Off, 1=On), bit 14: In-flight calibration flag (0=Off, 1=On), bit 15: Range flag (0=Low fields, high sensitivity, 1=High fields, low sensitivity), bit 16: Calibration flag (0=Old calibration, 1=New calibration), bit 17 ms added to sync correctly with IDPU, bit 18: 62.5 ms to 08 hz data (no longer used), bit 19: timestamp resync, the time stamp has been changed (i.e. th s glitch), bit 20: data rate resync, timestamps were calculated wit different Δt for mixed frequency data as the data rate was not synced properly so the records timestamps were calculated incorrectly, bit 21: derived from 32 hz Magnetometer data.
