LANL Daily-averaged electron fluxes from the Synchronous Orbit Particle Spectrometer (SOPA)

ResourceID
spase://NASA/NumericalData/LANL/Avg/SOPA/PT1D

Description

Daily-averaged electron fluxes from the Los Alamos
National Laboratory (LANL) Synchronous Orbit Particle Spectrometer
(SOPA) Daily averages from each geosynchronous satellite available
on any given day were merged into a single, uniform daily average.
Data span 22 September, 1989 through 31 December, 2009. Data are
log(flux) with flux in units of electrons /(cm^2.s.sr.keV). The
original SOPA data channels have been fit to a bi-Maxwelian spectra.
Fluxes are evaluated at virtual "channels" for fixed energies (i.e.
not for an energy passband or integral thresholds) [see 2010JA015735
_LANL_Data_and_Processing for details] (2010ja015735-readme.txt)

Caveats
"Only limited ‘cleaning’ of this data set is performed prior to daily averaging and merging. Any detected failures of the fitting procedure are first removed from the data archive prior to daily averaging. A small fraction of daily-averaged flux values appear to be unphysically high. A basic ‘cleaning’ is done by excluding any data points at which the evaluated daily average flux at more than four energies is over four standard deviations above the mean daily flux for each satellite. We stress that these unphysical values represent only a small fraction of the data set – many, but not all of these values are removed by the basic pre-processing applied here – and work is continuing to improve this data set. It should be noted that the very highest and lowest evaluated energies are extrapolated using the fits and may be less reliable than those energies within the detection limits of the instrument." (Electronic supplement to "On the relationship between relativistic electron flux and solar wind velocity: Paulikas and Blake Revisited”)
