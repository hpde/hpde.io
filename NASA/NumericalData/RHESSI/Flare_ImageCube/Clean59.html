RHESSI Flare ImageCubes - CLEAN59 Reconstruction

The RHESSI Consortium: UC Berkeley (USA); NASA Goddard Space Flight Center (USA); The Paul Scherrer Institut (Switzerland); Spectrum Astro (USA) (2022). RHESSI Flare ImageCubes - CLEAN59 Reconstruction [Data set]. Solar Data Analysis Center (SDAC).

ResourceID
spase://NASA/NumericalData/RHESSI/Flare_ImageCube/Clean59

Description

The RHESSI Flare Image Archive, located in the flare_images_v2 directory, contains image cube FITS files that can be used with RHESSI image object software, converted to map structures via hsi_fits2map, or read by any standard FITS reader.

The CLEAN59 algorithm is a variant of the CLEAN reconstruction technique that applies a specific set of parameters optimized for RHESSI imaging. Like CLEAN, it iteratively removes the instrument’s point-spread function from the initial Back Projection image, reducing artifacts and improving resolution. CLEAN59 uses a predefined configuration that balances image fidelity and computational efficiency, producing a cleaner and more accurate representation of solar flare sources.

CLEAN59 FITS files follow the naming convention hsi_imagecube_clean59_YYYYMMDD_HHMM_xxxxx.fits, where YYYYMMDD and HHMM represent the observation date and start time (UTC), xxxxx indicates the bin size identifier, and .fits denotes the standard Flexible Image Transport System format.
