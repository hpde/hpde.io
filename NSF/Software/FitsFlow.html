Description

FitsFlow is a browser-based application that processes solar FITS images by connecting to NASA/NOAA data services (HEK, JSOC, Helioviewer), synchronizing metadata and providing both in-browser visualization and bulk downloads of images, metadata, manifest files, and machine learning–ready ASDF outputs.

Acknowledgement
FitsFlow was supported by Georgia State University’s Solar Informatics and Data Mining Lab, inspired by HelioCloud, and made possible through data and services from Helioviewer, the Heliophysics Event Knowledgebase (HEK), JSOC, and the Python in Heliophysics Community (PyHC).
