Data point vector values which are a triplet of single precision floating point numbers which represent the X,Y,Z components of the electric field. These vectors are in one-to-one correspondence with the values in the TIME_TAG parameter. This data is computed by a ground spin-fit algorithm on the realtime electric field and represents one point every spin.
