Cassini Cruise Phase 1-Minute Magnetometer Data

ResourceID
spase://VSPO/NumericalData/Cassini/MAG/Cruise.Phase/PT1M

Description

Cassini cruise phase 1-minute-averaged magnetic field data taken during the interval between mid-1999 when near-continuous magnetic field observations were begun and the mid-2004 insertion of Cassini into Saturn orbit. There are no planetary encounter data included. The data are in RTN coordinates. The data are mostly from the fluxgate magnetometer (FGM). The table at https://spdf.gsfc.nasa.gov/pub/data/cassini/vhm.txt identifies the 52 days in 2000-2004 for which the data are solely from the vector helium magnetometer (VHM). Days not in the table contain only FGM data. The data were produced from raw (L1A) data at the PDS/PPI node using software provided by the Cassini MAG team, and employing the latest calibration available. PDS/PPI produced both 1-sec vectors and 1-min averages. The MAG team reports that while range-0 data are well calibrated for both FGM and VHM, higher range data need improvement.
