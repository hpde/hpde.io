Qsaybeh Magnetometer 1-hr Data in ASCII Flat Table Format

ResourceID
spase://VMO/NumericalData/WDC/Qsaybeh/Magnetometer/PT1H/ASCII_FORMAT

Description

Qsaybeh Magnetometer data, 1-hr time resoluion, in ASCII format. Magnetic Field Vector Element List: XYZF. Note that the list of magnetic field elements that appear in the data set may change over the course of time. For instance, many stations have shifted from DHZ to the XYZ representation of the magnetic field. The time history listed below summarizes changes in element content. A description of the WDC Data Exchange Format for CADENCE magnetic field data is included at the end of this data product description.
