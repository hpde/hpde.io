ARTEMIS P1 field-plasma merged data, near moon

ResourceID
spase://VMO/NumericalData/THEMIS/B/ESA.FGM/PT96S

Description

This data set contains 96s and 384s resolution
magnetic field and plasma ion data from the THEMIS-B ESA and FGM
instruments during the near moon (ARTEMIS) phase of the spacecraft
life (>~September 1, 2010). The data are at each resolution for
several mutually exclusive, contiguous hours during most days.
The field-plasma-merged data set was created at GSFC/Space Physics
Data Facility (SPDF) as part of the LunaSOX effort, from CDAWeb data
sets THB_L2_ESA (good quality, high angular resolution mode only)
and THB_L2_FGM. Geocentric spacecraft and moon orbit data from
SSCWeb were used to add geocentric and selenocentric spacecraft
position data to the merged data set records.
