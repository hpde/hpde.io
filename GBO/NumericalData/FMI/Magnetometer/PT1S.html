Description

The Finnish Meteorological Institute (FMI) provides archived and real-time 1-s magnetometer
data from twelve recording stations which are owned and maintained by FMI (KEV, MAS, KIL, IVA, MUO, PEL,
RAN, OUJ, MEK, HAN, NUR, and TAR). The real-time magnetometer data are provided with five minute update period.
