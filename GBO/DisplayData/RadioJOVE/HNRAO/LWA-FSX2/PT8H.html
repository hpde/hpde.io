Radio JOVE HNRAO LWA+FSX2 Spectrograph 15-30 MHz Dynamic Spectrogram Plots

ResourceID
spase://GBO/DisplayData/RadioJOVE/HNRAO/LWA-FSX2/PT8H

Description

Dynamic Spectrogram plots from output of the HNRAO LWA-type 4 phase crossed dipoles square array and FSX2 Spectrograph operating from 15-30 MHz. The plots are generated using the Radio-Sky Spectrograph software. The Display Cadence will vary but 8 hours is a reasonable maximum value for the timespan of a single file. Time is presented in Universal Time (UT).
