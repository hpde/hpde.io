Radio JOVE HNRAO TFD+SDRPlay Spectrograph 16-24 MHz Dynamic Spectrogram Plots

ResourceID
spase://GBO/DisplayData/RadioJOVE/HNRAO/TFD-SDRPlay/PT8H

Description

Dynamic Spectrogram plots from output of the HNRAO Terminated Folded Dipole and SDRPlay Spectrograph operating from 16-24 MHz. The plots are generated using the Radio-Sky Spectrograph software. The Display Cadence will vary but 8 hours is a reasonable maximum value for the timespan of a single file. Time is presented in Universal Time (UT). Both a RCP and a LCP dynamic spectrogram are produced as separate products.
