Description

This dataset contains 1 second measurements of the magnetic field vector from the Compact Spherical Coil (CSC) Fluxgate Vector Magnetometer and matrices describing rotation from J2000 reference system to stellar compass coordinate system and Earth Centered, Earth Fixed polar coordinates to J2000 reference system on the Oersted spacecraft.
