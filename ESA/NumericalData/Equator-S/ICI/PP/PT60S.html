Equator-S Ion Composition Instrument (ICI) Prime Parameters

ResourceID
spase://ESA/NumericalData/Equator-S/ICI/PP/PT60S

Description

This data set contains selected Key Parameters of the Ion Composition Instrument (ICI) onboard the Equator-S mission.
The following one-minute parameters are given: proton density, oxygen+ density, helium+ density, proton bulk velocity in both SCS and GSE coordinates, oxygen+ bulk velocity in both SCS (raw only) and GSE coordinates (final), proton temperature, oxygen+ temperature, helium+ bulk velocity in GSE coordinates (final), proton temperature tensor for one parallel and two perpendicular GSE components, oxygen+ temperature tensor for one parallel and two perpendicular GSE components, and helium+ temperature. The raw moments calculated onboard should only be used qualitatively for identifying regions and temporal variations. Quantitative analysis should be done with the final moments generated from telemetered 3D distributions.
